# Frontend Development Skill

## Instructions

1. **Page Structure**
   - Define clear page hierarchy
   - Use semantic HTML elements
   - Organize content sections logically
   - Ensure accessibility basics (labels, alt text)

2. **Component Design**
   - Break UI into reusable components
   - Keep components small and focused
   - Use props/state effectively
   - Follow consistent naming conventions

3. **Layout System**
   - Use Flexbox or Grid for layouts
   - Build responsive designs
   - Handle spacing and alignment consistently
   - Support mobile, tablet, and desktop views

4. **Styling**
   - Apply consistent color schemes and typography
   - Use utility classes or modular CSS
   - Avoid inline styles where possible
   - Maintain design consistency across pages

## Best Practices
- Mobile-first development
- Reusable and composable components
- Consistent spacing and typography
- Avoid deeply nested layouts
- Optimize for performance and readability
- Keep styles scalable and maintainable

## Example Structure
```html
<main class="page">
  <header class="header">
    <nav class="navbar">Navbar</nav>
  </header>

  <section class="content">
    <div class="card">
      <h2>Title</h2>
      <p>Description text</p>
      <button class="primary-button">Action</button>
    </div>
  </section>
</main>
